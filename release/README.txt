ShortsGak Release Notes
======================

[Quick Start]
1) Extract all files from this zip.
2) Run ShortsGak\ShortsGak.exe.
3) Enter VOD ID and keywords, then start analysis.

[Recommended Environment]
- Windows 10/11
- Internet connection (required for auto chatlog fetch)
- WebView2 Runtime

[Troubleshooting]
- If you see "Failed to fetch", close the app completely and run it again.
- If the issue continues, check this log file:
  ShortsGak\_internal\backend\logs\app.log
