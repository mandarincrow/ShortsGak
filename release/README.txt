ShortsGak 릴리스 노트
======================

[빠른 시작]
1) 이 ZIP 파일의 모든 내용을 추출합니다.
2) ShortsGak 폴더를 엽니다.
3) ShortsGak.exe를 실행합니다.
4) VOD ID와 키워드를 입력한 후 분석을 시작합니다.

[권장 환경]
- Windows 10/11
- 인터넷 연결 (자동 채팅 로그 가져오기 필요)

[문제 해결]
- "Failed to fetch" 메시지가 표시되면 앱을 완전히 종료한 후 다시 실행해 보세요.
- 문제가 계속되면 다음 로그 파일을 확인하세요:
  ShortsGak\resources\backend\logs\app.log
