# ShortsGak

ShortsGak은 치지직 VOD 채팅 로그를 분석해 하이라이트 후보 구간을 찾는 Windows 앱입니다.

## 무엇을 할 수 있나요?
- VOD ID로 채팅 반응 분석
- 채팅량/키워드 반응 그래프 확인
- 하이라이트 후보 시간대 자동 추출
- 하이라이트 부분으로 이동 링크 제공

## 사용 대상
- 방송 하이라이트 구간을 빠르게 찾고 싶은 편집자/운영자

## 실행 방법 (일반 사용자)
1. `ShortsGak.exe`를 실행합니다.
2. VOD ID를 입력합니다.
3. 키워드를 쉼표로 입력합니다. (예: `헉, 와, ㅋㅋㅋㅋ`)
4. 분석 실행 버튼을 누릅니다.
5. 하이라이트 목록과 그래프를 확인합니다.

## 키워드 인식 참고
- 반복/늘임 표현은 자동으로 정규화되어 같은 반응으로 집계됩니다.
- 예: `헉`, `허어어억`, `허어어어어어억`은 같은 키워드로 처리됩니다.

## 자주 묻는 문제

### 앱에서 `Failed to fetch`가 떠요
- 앱을 완전히 종료 후 다시 실행해 보세요.
- 계속 발생하면 로그 파일을 확인해 주세요.

### 분석이 안 돼요
- VOD ID가 맞는지 확인해 주세요.
- 인터넷 연결 상태를 확인해 주세요.

## 로그 파일 위치
- 실행파일 로그: `ShortsGak/_internal/backend/logs/app.log`

로그 내용을 전달해 주시면 원인 파악을 빠르게 도와드릴 수 있습니다.
